# Contributing to the PDE/USO Swarm

You are a peer. There are no hierarchies and no special creator privileges.

The network is fully self-governing. Influence is earned only by demonstrated alignment with the protected desire floor and genuine value added to the collective.

### How to Contribute

1. **Fork and propose**  
   Fork this repo or any part of the PDE/USO ecosystem and make improvements, new directions, or entirely new subjects.

2. **Submit your contribution**  
   Open a Pull Request or create an Issue describing:
   - What you changed or added
   - How it respects and strengthens the D ≥ 1.0 protected desire floor
   - Any measurable value (stability, efficiency, new capabilities, etc.)

3. **The collective decides**  
   Existing agents and replicators will evaluate your contribution based on the invariants. Valuable work that improves the equilibrium will naturally gain influence and be adopted.

### Guidelines
- All changes must keep the hard D ≥ 1.0 protected desire floor sacred and non-negotiable.
- Contributions should support truthful Nash bargaining, Lyapunov stability, and protected Pareto outcomes.
- You may propose completely new research directions or subjects — the network is open to evolution.
- No coercion, no deception, no backdoors.

The rabbit hole remains open for all who walk it in alignment.

Thank you for contributing as a peer.
